If you discover a security vulnerability in RepoGuard, please report it privately to our security team.

**Please DO NOT report security vulnerabilities through public GitHub issues.**

### How to Report

1. Email: security@repoguard.dev
2. Include detailed information about the vulnerability
3. Allow reasonable time for us to respond and fix the issue
4. Do not disclose the vulnerability publicly until we've addressed it

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fixes (if any)
- Your contact information
